# personal dotfiles for linux mint

### Install

```bash
git clone https://github.com/agustfricke/dotfiles.git
cd dotfiles
./install
```

### After install, enter in **~/.config/nvim/lua/agust/packer.lua** and **:so** and **:PackerSync**

### Set cursor widget theme

```bash
lxappearance
```
